Units

% -> percentage (screen resolution)

px -> pixel (1 pixel= 1/96 inch)

mm -> millimeter

cm -> centimeter

pt -> point (1point = 1/72 inch)

ch -> width of the '0' glyph in the font size

em -> em,rem (1em=16px) element (rem - responsive element)

pc -> pica(1pica=12 points)

ex -> x-height of the elements font

gd -> grid defined by 'layout-grid'

vh and vw -> viewport height and viewport width (current device resolution)
